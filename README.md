# ClipVault

ClipVault is a full-stack media URL organizer. It allows users to save video links, organize them with statuses and tags, search their saved clips, and prevent duplicate URLs from being saved.

## Project Goal

The goal of ClipVault is to help users manage large collections of saved video links. Instead of keeping hundreds of tabs open or losing links in browser history, users can save clips in one organized library.

## Features

- Add video/media URLs
- Normalize URLs before saving
- Prevent duplicate saved clips
- View saved clips
- Search saved clips
- Filter clips by status
- Edit clip details
- Delete clips
- Store data with a simple JSON file backend

## Tech Stack

### Frontend

- React
- Vite
- Axios
- CSS

### Backend

- Node.js
- Express
- CORS
- JSON file storage
